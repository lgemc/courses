# ECON 50 - Using Big Data to Solve Economic and Social Problems

## Overview
This course demonstrates how "big data" can address important social and economic problems. It introduces frontier research and policy applications, showing students how modern data science techniques intersect with economics.

## Core Topics

### Big Data Fundamentals
- What constitutes "big data"
- Data sources and collection methods
- Data quality and reliability
- Privacy and ethical considerations

### Economic Applications
- Labor market analysis
- Consumer behavior prediction
- Market efficiency studies
- Income inequality measurement

### Social Problem Solving
- Education outcomes
- Healthcare access and quality
- Criminal justice reform
- Urban planning and development

### Research Methods
- Causal inference with big data
- Machine learning techniques
- Natural experiments
- Randomized controlled trials

### Policy Applications
- Evidence-based policymaking
- Real-time policy evaluation
- Predictive analytics for intervention
- Cost-benefit analysis

### Tools and Techniques
- Statistical software introduction
- Data visualization
- Regression analysis
- Classification and prediction models

## Learning Objectives
- Understand the potential and limitations of big data
- Apply data analysis to economic and social questions
- Evaluate research using big data methods
- Develop skills in data-driven problem solving

## Applications
This course prepares students to:
- Use data to inform policy decisions
- Conduct empirical research in economics
- Evaluate claims based on data analysis
- Apply modern analytical tools to real-world problems

## Prerequisites
ECON 10A recommended; basic statistics helpful
